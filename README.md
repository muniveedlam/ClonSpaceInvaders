# Space Invaders - Proyecto Final en Java

Una recreación del clásico juego arcade 2D desarrollado completamente en Java utilizando la biblioteca gráfica Swing y AWT. Este proyecto está estructurado con Maven.

## 🚀 Características

* **Jugabilidad Clásica:** Defiende tu zona de las oleadas de alienígenas.
* **Movimiento Omnidireccional:** A diferencia del juego original, la nave puede moverse libremente por la pantalla (Arriba, Abajo, Izquierda, Derecha).
* **Dificultad Progresiva:** Al limpiar la pantalla de enemigos, la siguiente oleada aumenta en número de columnas y filas.
* **Enemigos Activos:** Los aliens tienen la capacidad de devolver el fuego de forma aleatoria.
* **Efectos Multimedia:** Gráficos estilo *pixel art* y efectos de sonido integrados para disparos, explosiones y fin del juego.
* **Escalado Dinámico:** El área de juego se adapta y mantiene su relación de aspecto al renderizarse.

## 📋 Requisitos Previos

Para ejecutar y compilar este proyecto necesitarás:

* [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/downloads/) (Versión recomendada: 17 o superior)
* [Apache Maven](https://maven.apache.org/)
